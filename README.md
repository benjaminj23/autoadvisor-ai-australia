# AutoAdvisor AI Australia

A Streamlit MVP for Australian used-car buyers.

## Features
- Personalised car recommendations
- Ownership-cost estimates
- Listing risk analysis
- PPSR readiness workflow
- Marketplace search links
- Buyer report download
- Human expert-help / partner lead request

## Local run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Community Cloud

Deploy from GitHub with `app.py` as the main file path.

## Disclaimer

Prototype only. Not legal, financial, insurance, mechanical, or vehicle-history advice.
